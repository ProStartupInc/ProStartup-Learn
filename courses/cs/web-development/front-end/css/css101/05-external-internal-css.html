CSS 101

5. Externí a interní CSS

Hlavní strana > Web > Front⁠-⁠End > CSS > CSS 101 > Lekce 5

V posledních třech lekcích jsme se učili o syntaxi, selektorech a do menší míry i o HTML. To vám je ale zatím kompletně k ničemu - neumíte to zatím implementovat do HTML dokumentu. To se ale v této lekci změní - řekneme si o externím, interním a řádkovém CSS.

Externí CSS

Externí CSS je vlastně samostatný soubor .css, který se připojí k HTML dokumentu za pomoci elementu link. Ten je potomkem head a měl by obsahovat atributy 'href' (který specifikuje cestu k souboru), 'rel' (určuje vztah mezi HTML dokumentem a připojovaným souborem) s hodnotou "stylesheet" a 'type' (říká prohlížeči, kterého typu vlastně připojovaný soubor je) s hodnotou "text/css"

    <link rel="stylesheet" type="text/css" href="nazevSouboru.css">

Externí stylesheety mají tu výhodu, že CSS je od HTML kompletně oddělené, což přispívá k lepší organizaci kódu. Navíc lze jeden použít na libovolný počet stránek, takže pokud celý váš web používá jeden stylesheet a potřebujete změnit nějaký styl, je to záležitost úpravy jen jednoho souboru.

Je také možné použít stylesheet z externího webu. To ale není zrovna nejspolehlivější - pokud se ten stylesheet náhodou stane nedostupným, vaše stránky ztratí veškeré stylování z něj.

Interní CSS

Interní CSS se používá hlavně v případě, kdy je nějaká stránka stylována jinak než ty ostatní nebo je potřeba přepsat nějaký styl bez zásahů v stylesheetu - má totiž vyšší prioritu než externí CSS. Vytváří se s elementem style, který je potomkem head.

    <style>
        p {
            color: red;
        }
    </style>

Řádkové CSS

Řádkové CSS (nazýváno také jako "inline") je CSS vepsané přímo do elementu za pomoci globálního atributu 'style'. Hodí se, když je jeden stylován jinak než zbytek a je zbytečné vytvářet pro něj novou třídu nebo ID. Navíc má vyšší prioritu než externí a interní CSS.

    <p style="color: red;">Příklad.</p>

Tento způsob by se neměl příliš používat, protože je nepřehledný.
