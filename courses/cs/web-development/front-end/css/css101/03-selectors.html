CSS 101

3. Selektory

Hlavní strana > Web > Front⁠-⁠End > CSS > CSS 101 > Lekce 3

V minulé lekci jsme si řekli, že selektory jsou prvním prvkem CSS pravidel a specifikují, ke kterým HTML elementům se vztahují. V této lekci si řekneme, jaké základní typy selektorů vlastně existují.

Typový selektor

Nejjednodušší typ selektorů, který vybírá elementy specifikovaného typu. Zapisuje se v nich jen název požadovaného elementu (tedy vše, co se nachází mezi počáteční a uzavírací špičatou závorkou počátečního tagu). Pravidlo v příkladě níže vykreslí všechny odstavce červené.

        p {
            color: red;
        }

Třídní selektor

Typ selektorů, který vybírá elementy na základě hodnoty jejich globálního atributu 'class'. Pokud se má pravidlo vztahovat na všechny elementy s jednou třídou nehledě na jejich typ, stačí jako selektor napsat tečku a hned po ní (bez mezery) název třídy. Příklad níže tedy obarví červeně všechny elementy, které mají třídu červená, nehledě na jejich typ.

        .červená {
            color: red;
        }

Pokud se má ale pravidlo vztahovat jen k jednomu typu elementů, před tečku v selektoru se musí napsat jeho název. Příklad níže obarví červeně všechny odstavce s třídou červená.

        p.červená {
            color: red;
        }

Identifikátorový selektor

Typ selektorů vybírající element na základě hodnoty jeho globálního atributu 'id'. Vzhledem k tomu, že identifikátor (zkráceně ID) musí být unikátní (neboli se v dokumentu objevit pouze u jednoho elementu), se tento typ selektorů zaměřuje jen na jeden specifický element. Má navíc větší prioritu než třídní.
Identifikátorové selektory se zapisuje stejně jako třídní, jen se místo tečky píše křížek #. Příklad níže obarví element s ID červená červeně.

        #červená {
            color: red;
        }

Stejně jako u třídních selektorů lze opět specifikovat, ke kterému typu elementů se bude ten identifikátorový vztahovat. Avšak vzhledem k tomu, že každé ID je unikátní, je tohle zbytečné.

        p#červená {
            color: red;
        }

Identifikátorové selektory mají vyšší prioritu než třídní. To znamená, že pokud jedno pravidlo s třídním selektorem říká, že má být element modrý a druhé pravidlo s identifikátorovým selektorem říká, že by měl být červený, prohlížeč se bude řídit tím identifikátorovým, i kdyby se nacházel na začátku stylesheetu před pravidlem se selektorem třídním.

        p#červená {
            color: red;
        }

        .modrá {
            color: blue;
        }

Skupinový selektor

Dejme tomu, že potřebujete, aby červenou barvu měly všechny odstavce, nadpisy h2 a elementy s třídou červená. Je možné vytvořit tři pravidla, která toho dosáhnou:

        p {
            color: red;
        }

        h2 {
            color: red;
        }

        .červená {
            color: red;
        }

Tento způsob se ale příliš zdlouhavý a zabírá zbytečně mnoho místa. Místo tří pravidel lze naštěstí vytvořit jen jedno, kde jsou všechny selektory vedle sebe, ale odděleny čárkou.

        p, h1, .červená {
            color: red;
        }

Selektorů lze takhle seskupit neomezeně mnoho.

Univerzální selektor

Univerzální selektor je selektor, který vybere všechny elementy v dokumentu. Zapisuje se se znakem hvězdičky *.

        * {
            color: red;
        }

Shrnutí

  • Typové selektory se vztahují k nějakému typu HTML elementů. Selektorem je název požadovaného elementu (např. p).
  • Třídní selektory se týkají nějaké HTML třídy. Selektorem je tečka, po které následuje název třídy (např. .červená). Před tečkou se může nacházet název elementu, na který selektor omezit (např. p.červená).
  • Identifikátorové selektory platí na nějaké specifické ID elementu. Mají vyšší prioritu než třídní. Selektorem je symbol křížku #, po kterém následuje samotné ID (např. #červená).
  • Skupinový selektor je vlastně více selektorů sebe; jsou jen odděleny čárkou (např. p, h2, #červená, .modrá).
  • Univerzálním selektorem je znak hvězdičky *. Vybírá všechny elementy v dokumentu.
