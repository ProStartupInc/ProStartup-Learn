CSS 101

5. <div> a <span>

Hlavní strana > Web > Front⁠-⁠End > CSS > CSS 101 > Lekce 5

V minulé lekci jsme si řekli, jak vybírat různé HTML elementy a aplikovat na ně styly. Existuje ale menší problém...Jak například stylovat jen určité sekce textu?

S tímto a dalšími problémy nám pomůžou HTML elementy <div> a <span>.

<span>

Pokud jste absolvovali na tomto webu kurz HTML 101, možná si pamatujete, že jedna z lekcí byla o blokových a řádkových elementech. <span> je elementem řádkovým, což znamená, že zabírá jen tolik místa, kolik potřebují a automaticky nevytváří pro sebe nové řádky jako elementy blokové.

Řádkové elementy se mohou nacházet uvnitř blokových. To znamená, že <span> může být potomkem <p> a obalit v něm nějaký text.

    <p>Tohle je nějaký text. <span;>Tento je obalen elementem &lt;span&gt;</span>, kdežto tento není.</p>

<span> je jedním ze dvou elementů, které defaultně nedělají absolutně nic a nemají nějaký sémantický význam...což to je přesně ten důvod, proč je perfektní pro stylování textu!

Pokud budete někdy potřebovat nějak ostylovat text, který nemá mít žádný sémantický význam - ať už změnit jeho barvu, podtrhnout ho, nebo změnit jeho velikost - nejlepší bude použít <span>. Je sice teoetický možné použít elementy jako <ins> nebo <strong>, ale ty všechny něco v HTML znamenají. Šlo by použít <b> jakožto element bez jakéhokoliv významu, ale proč to dělat, když je na stylování specificky určen <span>?

    <p>Tohle je nějaký text. <span style="color:red;">Tento je obalen elementem &lt;span&gt;</span>, kdežto tento není.</p>

Poznámka: O atributu 'style' si řekneme v příští lekci.

<div>

<div> je stejný jako <span>, jen je to místo řádkového elementu element blokový. Kvůli tomu se zatím může zdát méně užitečný než <span>, ale to se změní, až začneme probírat pozicování.

Kaskádové styly

Když už jsme u <div>, můžeme si rovnou říct o podstatě CSS.

Všechny styly se "dědí" z rodičovských elementů na potomky...

    <p style="color:red;">Tohle je nějaký text, který bude červeně. <span>Tento je obalen elementem &lt;span&gt; a přestože nemá sám o sobě žádné aplikované styly, je také červený</span>.</p>

...pokud není specifikováno jinak.

    <p style="color:red;">Tohle je nějaký text, který bude červeně. <span style="color:initial;">Tento je obalen elementem &lt;span&gt; který červený nebude</span>.</p>

Shrnutí

  • V HTML jsou dva elementy, které nic nedělají a nemají žádný sémantický význam, díky čemuž se mohou použít na stylování:
    • <span> je řádkový element, který se hlavně používá na obalení textu, jež je potřeba nějak ostylovat
    • <div> je blokový element. Více o něm v příštích lekcích.
  • Styly v CSS se dědí z rodičovských elementů na potomky
