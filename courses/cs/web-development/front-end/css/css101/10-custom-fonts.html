10. Vlastní fonty

Hlavní strana > Web > Front⁠-⁠End > CSS > CSS 101 > Lekce 10

Používat bezpečné fonty je často dostačující. Pokud chcete, aby váš web vypadal méně genericky, stačí využít písma z Google Fonts. Co když ale chcete použít nějaký font, který ve službě není? Nebo co když se chcete spoléhat jen sami na sebe a neriskovat, že služba bude mít výpadek nebo že ji Google z nějakého důvodu ukončí?

Spoléhat se na co nejméně jiných webů se v současné době navíc skvělou praktikou. V této lekci si ukážeme, jak na stránku přidat lokálně uložené fonty.

V této lekci si řekneme, jak na web přidat s pomocí CSS lokální fonty.

Stáhnutí fontu

Fonty se dají stáhnout na mnoha webových stránkách - včetně Google Fonts, který má tu výhodu, že veškerá písma tam jsou zdarma.

Stáhnutá písma budou většinou uložena ve formátu .ttf. Mnohé služby je nejdříve zkomprimují do souboru .zip, který budete muset nejdříve extrahovat. .ttf soubory by jste měli dát někam do složky svého webu.

Dávejte si pozor na autorská práva! Vždy zkontrolujte licenci fontu, který chcete použít.

Pravidlo @font-face

V CSS existuje několik speciálních pravidel (anglicky at-rules), které specifikují, jak se má CSS chovat. Jedním z nich je pravidlo '@font-face', se kterým lze specifikovat nové rodiny fontů. Zapisuje se jako normální pravidlo, jako selektor musí být ale místo elementu uvedeno '@font-face'. Mělo by se nacházet hned na začátku .css stylesheetu.

    @font-face {
        font-family: název fontu;
        src: cesta k souboru;
        font-weight: tloušťka fontu;
        font-style: styl fontu;
    }

Pravidlo by mělo obsahovat čtyři vlastnosti:

  • font-family: Specifikuje název písma, který se bude v CSS souboru používat. Nezáleží na tom, zda se shoduje se skutečným názvem fontu či ne.
  • src: Určuje umístění fontu, které se zapisuje stejným způsobem jako v HTML (viz HTML 101, lekce 7). Umístění musí být obklopeno uvozokami a musí se nacházet v CSS funkci 'url()'.
  • font-weight: Specifikuje tloušťku font. Jde specifikovat buď slovně (hodnotou může být buď normal nebo bold; první určuje normální tloušťku a druhá určuje tučné písmo(, nebo jakýmkoliv číslem od 1 do 1000 (tloušťka 400 je ekvivalentem normal a 700 zase bold).
  • font-style: Specifikuje kurzívu fontu. Má tři slovní hodnoty: normal (bez kurzívy), italic (s kurzívou) a oblique (s kurzívou, u které je možné úhel jejího sklonu nastavit - viz font-style).

Výsledné pravidlo může třeba vypadat takhle:

    @font-face {
        font-family: RobotoRegular;
        src: url("Roboto-Regular.ttf");
        font-weight: normal;
        font-style: normal;
    }

Použití nového fontu

Vámi specifikovaný font lze použít kdekoliv, ať už jen v jednom elementu, nebo na celé stránce. Stačí jen u nějakého pravidla použít vlastnost font-family a v ní jako hodnotu použít název písma, jež jste specifikovali ve vlastnosti font-family.

   Příklad

    <!DOCTYPE html>
    <html>
        <head>
            <link rel="stylesheet" href="https://fonts.googleapis.com/css?family=Open+Sans|Montserrat">
            <style>
                @font-face {
                    font-family: Roboto;
                    src: url("/fonts/Roboto-Regular.ttf");
                    font-weight: normal, serif-sans;
                    font-style: normal, serif-sans;
                }

                body {
                    font-family: Roboto, serif-sans;
                }
            </style>
        </head>
        <body>
            <p><strong>Lorem ipsum</strong> dolor sit amet. Praesent a vestibulum odio, <em>at gravida ipsum<em>.</p>
        </body>
    </html>

Fontů můžete specifikovat kolik chcete. Měli by jste ale mít na paměti, že si je bude muset uživatel stáhnout, takže to nepřežeňte. Nejlepší je použít dva až čtyři.
