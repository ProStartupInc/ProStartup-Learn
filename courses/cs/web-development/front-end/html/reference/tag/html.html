Současné HTML elementy

<html>

Hlavní strana > Web > Front⁠-⁠End > HTML > Elementy > <html>

<html> je element využívaný pro specifikaci celého HTML dokumentu, k jehož správnému fungování je povinný. Označuje se jako kořenový (root) - je totiž rodičem všech ostatních elementů na stránce.

Syntaxe

<html> je párový tag, jehož potomkem je jakýkoliv jiný element. V kódu se musí nacházet po <!DOCTYPE>, a to pouze jednou.

1
2
3
4
<!DOCTYPE html>
<html lang="hodnota" globální-atribut="hodnota">
    Obsah dokumentu
</html>

Atributy

Element má jeden vlastní atribut - 'xmlns'. Tento atribut specifikuje jmenný prostor XML, ke kterému dokument náleží. Výchozí hodnotou je http://www.w3.org/1999/xhtml, která bude případně dosazena v případě, že není specifikována. Atribut je povinný v dokumentech, které jsou parsovány XML parsery (což zahrnuje i XHTML).

<html> podporuje globální atributy. Měl by měl správně zahrnovat globální atribut 'lang', který v kontextu elementu určuje jazyk stránky. Sice to není povinné, ale jeho zahrnutí pomáhá SEO; čtečkám obrazovek, které mohou na základě jeho hodnoty vybrat výslovnost; překladačům, které mohou správně určit jazyk k přeložení; a podobně.

Vzhled

<html> lze v CSS stylovat. Defaultně je o blokový element.

1
2
3
html {
    display: block;
}

Využití

<html> má jediné využití, a to jako rodičovský element všech ostatních.

<!DOCTYPE html>
<html lang="cs">
    <head>
        <title>Základní stránka</title>
    </head>
    <body>
        <p>Příklad...</p>
    </body>
</html>
