Současné HTML elementy

<label>

Hlavní strana > Web > Front⁠-⁠End > HTML > Elementy > <label>

<label> je element používaný pro vytváření názvů pro vstupní pole ve formulářích.

Syntaxe

<label> je párový tag. Mezi tím počátečním a koncovým se nachází název pro pole.

1
<label for="hodnota" globální-atribut="hodnota">Název pole</label>

Atributy

<label> má jeden vlastní atribut - 'for'. Tento atribut specifikuje, ke kterému elementu vlastně má název patřit. Hodnotou by mělo být id požadovaného elementu. Atribut by se měl zahrnovat vždy, pokud není požadovaný element přímo potomkem <label>.

Element podporuje také globální atributy.

Vzhled

<label> lze v CSS stylovat. Defaultně jde o řádkový element.

1
2
3
label {
    display: inline;
}

Využití

<label> se používá na vytváření a přiřazování názvů následujícím elementům:

  • <button>
  • <input> (kromě <input type="hidden">)
  • <meter>
  • <output>
  • <progress>
  • <select>
  • <textarea>
1
2
3
4
<form>
    <label for="priklad">Jméno: </label>
    <input type="text" name="jmeno" id="priklad">
</form>

Přestože se nemusí <label> použít a jako název lze prostě vedle požadovaného elementu využít obyčejný text, není to příliš doporučeno. <label> nabízí totiž nad textem několik výhod:

  • Praktičtější pro uživatele: Text <label> se v HTML počítá k elementu, pro který je určen. To znamená, že kliknutí na ten text znamená vlastně kliknutí na element. To dělá například zaškrtávací formuláře mnohem praktičtější, zejména na mobilních zařízeních, kde uživatel není tak precizní jako na desktopu.
  • Přístupnost: Protože spojení textu s příslušným elementem je v kódu dokumentu jasně definováno a není jen vizuální, uživatelé čteček obrazovek si dokážou lépe interpretovat, jaký typ dat vlastně do formuláře vlastně zadat - čtečky totiž mohou obsah <label> přečíst spolu v kombinaci s daným polem.

<label> bez atributu 'for'

Existují celkem dva způsoby, jak v kódu napsat <label>. Ten první jsme si ukázali v příkladech výše - k propojení s příslušným elementem je potřeba specifikovat atribut 'for'.

1
2
<label for="priklad">Jméno: </label>
<input type="text" name="jmeno" id="priklad">

Ten atribut lze vynechat tím, že se požadovaný element vpíše mezi počáteční a koncový tag <label> - bude jeho potomkem.

1
2
3
4
<label>
    Jméno:
    <input type="text" name="jmeno" id="priklad">
</label>

Oba způsoby jsou stejně validní - záleží jen na vás, který z nich si vyberete.
