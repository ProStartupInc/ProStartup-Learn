Současné HTML elementy

<del>

Hlavní strana > Web > Front⁠-⁠End > HTML > Elementy > <del>

<del> (deleted) je párový tag používaný pro přeškrtávání již neplatného textu.

Syntaxe

<del> je párový tag. Mezi počáteční a koncový tag se dává text, jež má být přeškrtnut.

1
2
3
<dl>
    <dd atribut="hodnota" globální-atribut="hodnota">Obsah</dd>
</dl>

Atributy

<del> má dva vlastní atributy. Podporuje také ty globální.

Název Povinné? Popis
cite Ne Specifikuje odkaz, kde je vysvětleno, proč byl text označen za neplatný.
datetime Ne Specifikuje datum a čas, kdy byl text označen za neplatný.

Vzhled

<del> lze v CSS stylovat. Níže jsou defaultní CSS vlastnosti:

1
2
3
dd {
    display: inline;
    text-decoration: line-through;
}

Využití

<del> se používá k označování již neplatného nebo odstraněného textu. Má sémantický význam - pokud chcete přeškrtnout text jenom jako dekoraci, použijte <s>.

1
<p>HTML je <del>programovací</del> značkovací jazyk.</p>
