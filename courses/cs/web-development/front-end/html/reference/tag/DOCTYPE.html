Současné HTML elementy

<!DOCTYPE>

Hlavní strana > Web > Front⁠-⁠End > HTML > Elementy > <!DOCTYPE>

<!DOCTYPE> je instrukce prohlížeči, která mu říká, že má .html soubor interpretovat jako .html soubor. Také specifikuje verzi HTML, kterou soubor používá. Nejde o HTML element!

Syntaxe

Zapisování <!DOCTYPE> je v současných verzích HTML velmi jednoduché. Nemusí se navíc dodržovat velikost písmen, což znamená, že všechno, co je v příkladu níže, je kompletně validní:

1
2
3
4
<!DOCTYPE html>
<!doctype html>
<!dOcTyPe hTmL>
<!DOCTYPE HTML>

<!DOCTYPE> musí být ale správně zapsán na to, aby fungoval. Níže jsou podmínky, které se musí dodržet:

  • Správné umístění: <!DOCTYPE> musí být vždy zapsán na prvním řádku HTML dokumentu.
  • Jen jednou: Většina moderních prohlížečů bude ignorovat všechny ostatní deklarace <!DOCTYPE> kromě té první. U některých se ale mohou vyskytnout vykreslovací potíže, nebo se mohou dokonce přepnou do tzv. nekompatibilního módu (viz sekce Invalidní <!DOCTYPE>).

Atributy

Protože jde spíše o instrukci prohlížeči než o normální HTML element, <!DOCTYPE> nemá žádné atributy; i přesto by ale někteří mohli říct, že 'html' je booleanovským atributem.

Vzhled

<!DOCTYPE> není pro uživatele viditelný a nijak neovlivňuje rozložení stránky. Nelze jej stylovat v CSS.

Invalidní <!DOCTYPE>

Když je deklarace <!DOCTYPE> invalidní nebo chybí, může to způsobit řadu potíží.

  • Nekompatibilní mód: Některé prohlížeče se mohou přepnout do nekompatibilního módu, který může způsobit mnoho potíži ve vzhledu stránky. Tento mód je totiž hlavně určen pro weby z dob, kdy byly hlavními prohlížeči na trhu Netscape a Internet Explorer. Stručně řečeno - tento mód není určen pro současné weby.
  • Nesprávné CSS: S invalidním <!DOCTYPE> mohou být CSS vlastnosti a box model interpretovány špatně, což může narušit rozložení stránky.
  • Nekonzistence: Každý prohlížeč si může stránku bez validního <!DOCTYPE> interpretovat jinak. To může způsobit nekonzistentní vzhled stránky mezi různými prohlížeči.
  • Kompatibilita: Bez validního <!DOCTYPE> nemusí fungovat některé funkce.
  • SEO: Invalidní <!DOCTYPE> může mít za následek horší umístění stránky ve vyhledávacích výsledcích.
  • Přístupnost: Invalidní <!DOCTYPE> může udělat stránku nepřístupnou pro uživatele s různými postiženími, protože čtečky obrazovek nemusí být schopny správně interpretovat její obsah.
