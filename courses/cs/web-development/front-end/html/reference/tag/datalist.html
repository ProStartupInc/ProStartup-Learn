Současné HTML elementy

<datalist>

Hlavní strana > Web > Front⁠-⁠End > HTML > Elementy > <datalist>

<datalist> je párový tag používaný pro vytváření nepovinných, doporučených možností pro pole ve formulářích.

Syntaxe

<datalist> je párový tag.

1
2
<input id="hodnota">
<datalist id="hodnota" atribut="hodnota" globální-atribut="hodnota"></colgroup>

Atributy

<datalist> podporuje jen globální atributy. Nejdůležitějším z nich je bezesporu 'id', jehož hodnota se dá použít k propojení elementu s požadovaným polem ve formuláři.

Vzhled

<dtalist> lze sice v CSS stylovat, ale je to kompletně zbytečné. Velká část CSS vlastností u tohoto elementu nefunguje. Defaultně nemá žádný vzhled.

1
2
3
datalist {
    display: none;
}

Využití

<datalist> se používá k vytváření doporučených možností u typů formulářových polí, kam může uživatel zadat libovolnou hodnotu. Lze ho použít v kombinaci s většinou typů <input> - je potřeba jen dát elementu <datalist> atribut 'id' a elementu <input>list', přičemž oba atributy budou mít stejné hodnotami.

Možnosti samotné se vytváří s potomky <option>.

U textových formulářů

U textových formulářů se možnosti automaticky ukážou po kliknutí na formulářové pole. Když uživatel klikne (nebo jen přejede myší) na nějaké pole propojené s <datalist>, v mnoha prohlížečích se uvnitř toho pole ukáže obrácený trojúheleník - indikátor toho, že jsou nějaké nepovinné možnosti dostupné. Jako textové formuláře se v tomto kontextu berou:

  • <input type="email">
  • <input type="number">
  • <input type="search">
  • <input type="tel">
  • <input type="text">
  • <input type="url">

U časových formulářů

U časových formulářů se možnosti z <datalist> obvykle ukážou v menu, kde se vybírá datum a/nebo čas. Jako časové formuláře se berou:

  • <input type="date">
  • <input type="datetime-local">
  • <input type="month">
  • <input type="time">
  • <input type="week">

U <input type="range">

U <input type="range"> se hodnoty specifikované v <datalist> zobrazí jako body v rozmezí, které bude moct uživatel vybrat snadněji a precizněji.
