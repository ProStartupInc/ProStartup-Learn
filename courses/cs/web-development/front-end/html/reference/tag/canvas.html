Současné HTML elementy

<canvas>

Hlavní strana > Web > Front⁠-⁠End > HTML > Elementy > <canvas>

<canvas> je párový tag používaný pro vytvoření "plátna", na které se s pomocí skriptování v JavaScriptu mohou vytvářet animace. Lze také použít jako kreslící plocha, na kterou bude moct uživatel kreslit.

Syntaxe

<canvas> je párový tag. Mezi ten počáteční a koncový se může dát text, který se ukáže uživateli v případě, že element není v jeho prohlížeči podporován.

1
<canvas atribut="hodnot" globální-atribut="hodnota">Text pro případ, že element není podporován</canvas>

Atributy

<canvas> má několik vlastních atributů. Kromě nich podporuje i ty globální.

Název Povinné? Popis
height Ano Specifikuje v pixelech, jak vysoká je kreslící plocha elementu.
moz-opaque Ne Atribut specifický jen k prohlížečům založených na Mozzile. Specifikuje, zda plátno podporuje průhlednost (defaultně ano).
width Ano Specifikuje v pixelech, jak široká je kreslící plocha elementu.

Vzhled

<canvas> lze v CSS stylovat. Defaultně nemá žádný vzhled - bez trochy CSS ho tak není schopné odlišit od zbytku stránky.

CSS vlastnosti sice mohou ovlivnit viditelnou šířku a výšku elementu, ale nemůžou ovlivnit šířku a výšku samotné kreslící plochy - k tomu jsou potřeba atributy 'height' a 'width'.

Využití

<canvas> nemá defaultně žádnou funkcionalitu - je potřeba použít JavaScript. Z toho důvodu si tu neukážeme, jak tento element použít.
