Podle Wikipedie je moře...

...velká plocha slané či brakické vody, spojená se světovým oceánem, a také s řekami. Někdy může být jako moře tradičně, ale nepřesně označeno též velké vnitrozemské (zpravidla slané) jezero, kterému schází přirozený odtok (např. Kaspické moře nebo Mrtvé moře). Zde však není splněna jedna ze základních vlastností, totiž spojitost s ostatními moři.
