Současné HTML elementy

<em>

Hlavní strana > Web > Front⁠-⁠End > HTML > Elementy > <em>

<em> (emphasis) je párový tag používaný pro vytváření kurzívy u textu, na který se má klást důraz.

Syntaxe

<em> je párový tag. Mezi počáteční a koncový tag patří jakýkoliv text, na který má být nějak kladen důraz.

1
2
3
<em globální-atribut="hodnota">Text s kladeným důrazem</em>

Atributy

<em> podporuje jen globální atributy.

Vzhled

<em> lze v CSS stylovat. Defaultně vypadá jako obyčejný text s kurzívou.

1
2
3
4
dt {
    display: inline;
    font-style: italic;
}

Využití

<em> se používá na označování slovnebo fráze, na kterou má být kladen důraz. Neměl by se používat jen stylově - na to existují CSS vlastnosti a <i>. Na důležitý text je tu <strong>.

1
<p>HTML <em>není</em> programovací jazyk - je to jazyk <em>značkovací</em>.</p>
