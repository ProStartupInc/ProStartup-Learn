Současné HTML elementy

<figure>

Hlavní strana > Web > Front⁠-⁠End > HTML > Elementy > <figure>

<figure> je element využíván pro obalování samostatného obsahu - neboli obsahu, který není k dokumentu hlavní, ale je v něm zmíněn a jeho pozice v něm nemění jeho význam.

Syntaxe

<figure> je párový tag.

1
2
3
<figure>
    Obsah.
</figure>

Atributy

<figure> podporuje pouze globální atributy.

Vzhled

<figure> lze v CSS stylovat. Defaultně jde o blokový element.

Využití

<figure> se používá na obalování obsahu, který sice souvisí s ostatním obsahem HTML dokumentu, ale není příliš důležitý a může být v něm umístěn kdekoliv bez toho, aby nastala nějaká změna významu. Do takového obsahu se například mohou počítat fotografie z výletu na cestovním blogu, tabulka se seznamem zaměstnanců, a podobně.

2
3
4
5
<figure>
    <img src="/graphics/ocean-waves-01.jpg" height="100">
</figure>
<p>Podle Wikipedie je moře...</p>
<blockquote>...velká plocha slané či brakické vody, spojená se světovým oceánem, a také s řekami. Někdy může být jako moře tradičně, ale nepřesně označeno též velké vnitrozemské (zpravidla slané) jezero, kterému schází přirozený odtok (např. Kaspické moře nebo Mrtvé moře). Zde však není splněna jedna ze základních vlastností, totiž spojitost s ostatními moři.</blockquote>

S <figcaption>

V kombinaci s <figure> se dá použít jeho potomek .

<figcaption> se používá k vytváření titulků <figure>, které mohou např. popisovat jeho obsah nebo obsahovat zdroj, ze kterého obsah je. Více informací na jeho stránce.

    <figcaption>
         <p><i>Kombinace modré a červené</i></p>
         <p>Zdroj: <cite>ProStartup Learn</cite></p>
    </figcaption>
</figure>
