Současné HTML elementy

<b>

Hlavní strana > Web > Front⁠-⁠End > HTML > Elementy > <b>

<b> je párový tag používaný pro vytváření nesémantického tučného textu.

Syntaxe

<b> je párový tag. Mezi tím počátečním a koncovým se nachází text, jež má být ztučněn. Nemusí mít žádného rodiče (kromě <body>), ale často se používá jako potomek <p>.

1
<b globální-atribut="hodnota">Obsah</b>

Atributy

<b> nemá žádné vlastní atributy, ale podporuje ty globální.

Vzhled

<b> lze v CSS stylovat.

1
2
3
4

Využití

<b> by se měl využívat na text, který je potřeba zvýraznit, ale nemá žádnou důležitost. Podle oficiální HTML specifikaci by se měl používat jako poslední možnost; před jeho použitím by se mělo zvážit, zda není lepší použít nějaký jiný element, jako třeba <strong> nebo <em>.

1
<p>Tento text není tučný...<b>ale tento je!</b></p>
