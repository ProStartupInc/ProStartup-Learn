Současné HTML elementy

<li>

Hlavní strana > Web > Front⁠-⁠End > HTML > Elementy > <li>

<li> je element využívaný pro vytváření položek do číslovaných (<ol>) a nečíslovaných seznamů (<ul>) a nabídek (<menu>). Každá položka může obsahovat cokoliv, od textu přes obrázky po další seznamy.

Syntaxe

<legend> je párový tag. Mezi tím počátečním a koncovým se nachází obsah titulku pro rodičovský <fieldset>. Vždy by měl být prvním potomkem <fieldset>!

1
2
3
<fieldset>
    <legend globální-atribut="hodnota">Titulek</legend>
</fieldset>

Atributy

<legend> podporuje pouze globální atributy.

Vzhled

Využití

Element se používá pro vytváření titulku/nadpisu u <fieldset>, který seskupuje skupinu souvisejících formulářových polí. Titulek by měl s obsahem <fieldset> souviset.

        <input type="text" name="jmeno" id="priklad1"><br><br>
        <label for="priklad2">Příjmení: </label>
        <input type="text" name="prijmeni" id="priklad2"><br><br>
        <label for="priklad3">Věk:&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;</label>
        <input type="number" name="vek" id="priklad3" max="122"><br><br>
        <label for="priklad4">Oblíbená barva: </label>
        <input type="color" name="barva" id="priklad4"><br><br>
        <input type="submit">
    </fieldset>
</form>
