Současné HTML elementy

<header>

Hlavní strana > Web > Front⁠-⁠End > HTML > Elementy > <header>

<header> (neplést s <head>) je HTML element používaný pro vytvoření hlavičky stránky, ve které se nachází například nadpisy.

Syntaxe

<header> je párový tag. Jeho potomky by neměl být element <footer> nebo další elementy <header>.

1
2
3
<header>
    Obsah.
</header>

Atributy

<header> podporuje pouze globální atributy.

Vzhled

<header> je možné v CSS stylovat. Defaultně jde o blokový element.

1
2
3
header {
    display: block;
}

Využití

<header> je určen hlavně na obklopení nadpisu <h1> nebo navigace. Může ale obsahovat například i loga, pole vyhledávání, a podobně.

1
2
3
4
<header>
    <h1>Co je example.com?</h1>
</header>
<p><a href="https://example.com">example.com</a> je webová stránka určená různé dokumentace, kde je potřeba jako příklad použít nějakou webovou adresu.</p>
