Současné HTML elementy

<kbd>

Hlavní strana > Web > Front⁠-⁠End > HTML > Elementy > <kbd>

<kbd> je element využívaný pro zvýrazňování textu, který má reprezentovat uživatelský vstup (ať už třeba klávesnicí nebo hlasovými příkazy).

Syntaxe

<kbd> je párový tag. Mezi tím počátečním a koncovým se nachází text, jež má být vyznačen.

1
<kbd globální-atribut="hodnota">Text</kbd>

Atributy

<kbd> podporuje pouze globální atributy.

Vzhled

<kbd> lze v CSS stylovat. Defaultně jde o řádkový element, přičemž text mezi počátečním a koncovým tagem je (alespoň ve většině prohlížečů) vykreslen v neproporciálním (monospace) písmu.

1
2
3
4
kbd {
    display: inline;
    font-family: monospace;
}

Využití

<kbd> se používá na vyznačování uživatelského vstupu zmíněného v textu, což se hodí nejvíce třeba do nějakých dokumentací aplikací. V závislosti na jeho potomcích nebo rodičích může mít více významů:

  • <kbd> uvnitř <kbd>: Element reprezentuje například specifickou klávesu na klávesnici nebo hlasový příkaz bez jakéhokoliv spojení s nějakým programem
  • <kbd> uvnitř <samp>: Element reprezentuje vstup tak, jak byl interpretován programem
  • <samp> uvnitř <kbd>: Element reprezentuje vstup, jež je závislý na výstupu nějakého programu (například vyvolání položky v nějaké nabídce)

Tyto významy ale můžete klidně ignorovat a na označení jakéhokoliv vstupu prostě použít jen jeden <kbd> bez jakýchkoliv potomků nebo rodičů <samp>.
