Současné HTML elementy

<dl>

Hlavní strana > Web > Front⁠-⁠End > HTML > Elementy > <dl>

<dl> (definition list) je párový tag používaný k vytváření definičních seznamů.

Syntaxe

<dl> je párový tag. Mezi počáteční a koncový tag patří elementy <dt> a <dd>.

1
2
3
4
<dl globální-atribut="hodnota">
    <dt>Nepovinné<dt>
    <dd>Nepovinné<dd>
</dl>

Atributy

<dl> podporuje pouze globální atributy.

Vzhled

<dl> lze v CSS stylovat. Níže jsou defaultní CSS vlastnosti:

    margin-top: 1em;
    margin-bottom: 1em;
    margin-left: 0;
    margin-right: 0;
}

Využití

<dl> se používá na vytváření definičních seznamů. Funguje jako rodičovský element pro <dt> a <dd>.

1
2
3
4
5
5
6
7
<dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language; značkovací jazyk používaný ve web designu</dd>
    <dt>CSS</dt>
    <dd><b>C</b>ascading <b>S</b>tyle <b>S</b>heets; jazyk používaný ke stylování HTML dokumentů</dd>
    <dt>JavaScript</dt>
    <dd>Programovací jazyk používaný pro vytváření interaktivích webových stránek</dd>
</dl>
