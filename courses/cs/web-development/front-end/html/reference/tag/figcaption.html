Současné HTML elementy

<figcaption>

Hlavní strana > Web > Front⁠-⁠End > HTML > Elementy > <figcaption>

<figcaption> je párový tag používaný pro vytváření popisků pro <figure>.

Syntaxe

<fieldset> je párový tag. Musí být buď prvním, anebo posledním potomkem <figure>.

1
2
3
4
5
<figure>
    <figcaption atribut="hodnota" globální-atribut="hodnota">
        Obsah.
    </figcaption>
</figure>

Atributy

<figcaption> podporuje pouze globální atributy.

Vzhled

<figcaption> lze v CSS stylovat. Defaultně jde o blokový element.

1
2
3
details {
    display: block;
}

Využití

<figcaption> se používá na vytváření titulku nebo popisu obsahu rodičovského elementu <figure>. Může také obsahovat dodatečné informace např. o původu obsahu.

1
2
3
4
5
6
7
<figure>
    <img src="/graphics/priklad.webp">
    <figcaption>
         <p><i>Kombinace modré a červené</i></p>
         <p>Zdroj: <cite>ProStartup Learn</cite></p>
    </figcaption>
</figure>
