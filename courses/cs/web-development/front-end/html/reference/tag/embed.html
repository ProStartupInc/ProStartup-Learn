Současné HTML elementy

<embed>

Hlavní strana > Web > Front⁠-⁠End > HTML > Elementy > <embed>

<embed> je nepárový tag používaný pro vkládání externího obsahu do HTML dokumentu, přičemž onen obsah je většinou obstarán prohlížečovými plug-iny nebo jinými externími aplikacemi.

Syntaxe

<embed> je nepárový tag.

1
<embed atribut="hodnota" globální-atribut="hodnota">

Atributy

<embed> podporuje několik vlastních atributů, které jsou jmenovány v tabulce níže. Podporuje také globální atributy.

Název Povinné? Popis
height Ne Specifikuje výšku elementu v pixelech.
src Ano Specifikuje cestu k souboru, který má být přes <embed> vložen.
type Ne Říká prohlížeči, jaký typ souboru je vlastně vkládán.
width Ne Specifikuje šířku elementu v pixelech.

Vzhled

<embed> lze v CSS stylovat. Defaultně nemá žádný vzhled. Pokud je přes něj vložen nějaký soubor, má stejné rozměry jako onen soubor.

Alternativy

Pokud je to možné, <embed> byste se měli vyhnout a místo toho použít nějaký jiný element - mnoho prohlížečů totiž ruší nebo již zrušilo podporu pro různé plug-iny, tak <embed> nefunguje všude. Níže jsou alternativy k němu:

  • Obrázky: <img> nebo <picture>
  • Audio: <audio>
  • Videa: <video>
  • HTML dokumenty: <iframe>

Využití

Přes <embed> lze vkládat velké množství typů multimédií. Avšak před tím, než ho použijete, byste se měli zamyslet nad tím, zda ho nelze nahradit nějakým jiným elementem.

Obrázky

1
<embed type="image/webp" src="/graphics/priklad.webp">

Audio

1
<embed type="audio/ogg" src="/sounds/heavy-thunder-01.ogg">

Videa

1
<embed type="video/webp" src="/videos/ocean-waves-01.webp">

HTML dokumenty

1
<embed type="text/html" src="/index-cs.html">
