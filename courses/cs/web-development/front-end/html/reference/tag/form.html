Současné HTML elementy

<form>

Hlavní strana > Web > Front⁠-⁠End > HTML > Elementy > <form>

<form> je element používaný pro vytváření formulářů.

Syntaxe

<form> je párový tag.

1
2
3
<form>
    Obsah.
</form>

Atributy

<form> má poměrně velké množství vlastních atributů. Podporuje také globální atributy.

Název Povinné? Popis
accept-⁠charset Ne Specifikuje, v jakých znakových sadách jsou hodnoty na server příjmány.
action Ano Specifikuje URL, které zpracuje data odeslaná z formuláře.
autocapitalize Ne Nestandartní atribut vztahující se pouze na iOS verzi Safari. Určuje, jak budou automaticky zvětšována písmena v hodnotách odeslaných na server (např. začátku věty nebo na začátku každého slova)
autocomplete Ne Určuje, zda bude prohlížeč uživateli navrhovat během zadávání hodnoty, které již dříve zadal v podobných formulářích.
enctype Ne ???.
method Ne Určuje, jaká HTTP metoda bude během odesílání dat použita.
name Ano Určuje název formuláře, podle kterého bude možné identifikovat v databázích, z jakého formuláře byla data odeslána.
novalidate Ne Booleanovský atribut. Pokud je přítomen, data odeslaná z formuláře nebudou validována
rel Ne ???.
target Ne Specifikuje, kam uživateli ukázat odpověď po odeslání formuláře.

Vzhled

<form> lze v CSS stylovat. Defaultně jde o blokový element.

1
2
3
4
form {
    display: block;
    margin-top: 0;
}

Využití

<form> se používá na vytváření formulářů. K tomu je ale potřeba ještě potomek <input>, který specifikuje jednotlivá pole, do kterých lze hodnoty zadat.

1
2
3
4
5
<form>
    <label for="formular">Jméno: </label>
    <input type="text" id="formular" name="jmeno"><br><br>
    <input type="submit">
</form>
