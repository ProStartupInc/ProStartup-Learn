Webové technologie

Webové technologie se dají obecně rozdělit na dva typy.

Front-End

Front-End technologie jsou ty, které jsou zpracovávány na straně klienta - tedy na uživatelově prohlížeči. Jsou zodpovědné za prezentaci stránky a za její obsah. Existují tři hlavní, které by měl znát každý front-end vývojář.

HTML

HTML je značkovací jazyk, který se stará o obsah stránky.

CSS

CSS je jazyk používaný pro stylování stránek. Stará se o jejich vzhled, rozložení, atd.

JavaScript

JavaScript se stará o to, aby byly stránky nějak interaktivní.

Back-End

Back-End technologie jsou ty, které běží na serverech. Starají se třeba o databáze, atd. Příkladem je třeba PHP nebo Node.js.
