Současné HTML elementy

<details>

Hlavní strana > Web > Front⁠-⁠End > HTML > Elementy > <details>

<details> je párový tag používaný k vytváření interaktivních widgetů, které po kliknutí expandují a odhalí bližší informace.

Syntaxe

<details> je párový tag. Mezi počáteční a koncový tag se dává obsah, jež by se měl ve widgetu nacházet a potomek <summary>, který pro něj poskytuje titulek/nadpis (pokud není <summary> specifikován, prohlížeč ukáže uživateli svůj vlastní, defaultní text).

2
3
4
<details atribut globální-atribut="hodnota">
    <summary></summary>
    <Obsah.
</details>

Atributy

<details> má jeden vlastní, booleanovský atribut - open. Pokud je přítomen, tak bude widget defaultně ve svém expandovaném stavu. Element také podporuje globální atributy.

Vzhled

<details> lze v CSS stylovat. Defaultně jde o blokový element.

1
2
3
details {
    display: block;
}

Využití

<details> se používá k vytváření widgetů, které se po kliknutí rozloží a odhalí dodatečné informace. To se hodí na stránky s velkým množstvím obsahu.

Widget je defaultně v zavřeném (nebo neexpandovaném) stavu, kdy se zobrazuje jen jeho titulek, před kterým se nachází ikona trojúhelníku. V otevřeném (nebo expandovaném) stavu se pod titulkem objeví obsah widgetu a ikona trojúhelníku se obrátí o 90 stupňů doleva.
