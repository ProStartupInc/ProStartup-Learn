<bdo>

Hlavní strana > Web > Front⁠-⁠End > HTML > Elementy > <bdo>

<bdo> (Bidirectional override) je párový tag používaný pro změnu směru nějakého textu.

Syntaxe

<bdi> je párový tag. Mezi tím počátečním a koncovým se nachází text, jehož směr psaní má být změněn.

1
<bdo dir="hodnota" globální-atribut="hodnota">Obsah</bdo>

Atributy

<bdi> nemá žádné vlastní atributy, ale podporuje ty globální.

Vždy by se měl do <bdi> vložit globání atribut 'dir', který specifikuje směr, kterým bude text vypsán. Bez něj tento element nedává smysl. Také by se neměla používat hodnota "auto".

Vzhled

<bdi> lze v CSS stylovat. Níže jsou defaultní CSS vlastnosti.

1
2
3
4
bdo {
    display: inline;
    unicode-bidi: bidi-override;
}

Využití

<bdi> by se měl využívat v případě, že je třeba změnit směr nějakého textu. Pokud není jasné, jakým směrem by ten text měl být vypsán, použijte <bdi>.

1
2
<p>Následující text je zleva doprava: Vítejte.</p>
<p>Následující text je zprava doleva: <bdo dir="rtl">Vítejte.</bdo></p>
