Současné HTML elementy

<fieldset>

Hlavní strana > Web > Front⁠-⁠End > HTML > Elementy > <fieldset>

<fieldset> je párový tag používaný pro seskupování souvisejících prvků formulářů.

Syntaxe

<fieldset> je párový tag. Je potomkem <form>.

1
2
3
4
5
<form>
    <fieldset atribut="hodnota" globální-atribut="hodnota">
        Obsah.
    </fieldset>
</form>

Atributy

<embed> podporuje několik vlastních atributů. Podporuje také globální atributy.

Název Povinné? Popis
disabled Ne Booleanovský atribut. Pokud je přítomen, uživatel nebude moct interagovat s potomky elementu.
form Ano Specifikuje, ke kterému formuláři element vlastně patří. Nedědí se na potomky - musí se použít zvlášť na ně a na <fieldset>.
name Ne Určuje název spojený se skupinou.

Využití

<fieldset> se používá na seskupování prvků formulářů, které spolu nějak souvisí (nejčastěji tématicky; např. pole vztažena na osobní informace). To je obzvláště užitečné v dlouhých formulářích.

<form>
    <fieldset>
         <label for="jmeno">Jméno: </label>
         <input type="text" id="jmeno" name="jmeno"><br>
         <label for="prijmeni">Příjmení: </label>
         <input type="text" id="prijmeni" name="prijmeni">
    </fieldset>
<form>

S <legend>

<fieldset> je možné použít s elementem <legend>, který je jeho potomkem. <legend> se používá k vytváření titulku/nadpisu pro sekci obklopenou <fieldset>.

         <label for="jmeno">Jméno: </label>
         <input type="text" id="jmeno" name="jmeno"><br>
         <label for="prijmeni">Příjmení: </label>
         <input type="text" id="prijmeni" name="prijmeni">
    </fieldset>
<form>
