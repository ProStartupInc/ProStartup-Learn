Současné HTML elementy

<ins>

Hlavní strana > Web > Front⁠-⁠End > HTML > Elementy > <ins>

<ins> je element využívaný pro podtrhování textu v dokumentu, který byl přidán po jeho zveřejnění.

Syntaxe

<ins> je párový tag. Mezi tím počátečním a koncovým se nachází text, jež má být podtržen. Neměl by přesahovat přes hranice odstavců.

1
<ins atribut="hodnota" globální-atribut="hodnota">Text</ins>

Atributy

<ins> podporuje pouze dva vlastní atributy, přičemž ani jeden z nich není povinný.

Název Povinné? Popis
cite Ne Odkaz na stránku/dokument, kde je vysvětlen důvod, proč byl text přidán.
datetime Ne Poskytuje datum a čas změny.

Element také podporuje globální atributy.

Vzhled

<ins> lze v CSS stylovat. Defaultně jde o řádkový element, který vypadá - až na podtržení - jako normální text.

1
2
3
4
ins {
    display: inline;
    text-decoration: underline;
}

Využití

<ins> se používá na vyznačování textu, který byl do dokumentu přidán až po jeho zveřejnění. Využívá se někdy v kombinaci s <del>, který slouží k přeškrtávání již neplatného nebo prostě nepotřebného textu.

1
<p>Moravský Krumlov je jihomoravské město nacházející se poblíž Ivančic. Jeho populace <del>je kolem 5 tisíc</del> <ins>činila k roku 2019 kolem 5 700</ins> obyvatel.</p>

Alternativy

Pokud nechcete vyznačovat přídavky do dokumentu a místo toho podtrhnout text jen dekorativně, můžete použít CSS vlastnost text⁠-⁠decoration v kombinaci s elementem <span>. Pokud vám ale na podtržení nezáleží a chcete prostě jen nějak vyznačit (ať už sémanticky nebo ne) text, pravděpodobně budete lepší použít jeden z následujících elementů:

  • <b>: Tučné vyznačení textu, který nemá žádnou důležitost.
  • <em>: Kurzíva používaná na sémantické zdůraznění označeného textu.
  • <i>: Kurzíva používaná pro zvýraznění textu v jiném tónu (například odborné termíny).
  • <mark>: Zvýraznění textu, který je relevantní i v jiném kontextu.
  • <strong>: Sémantické zdůraznění důležitosti označeného textu, který bude v dokumentu zobrazen tučně.
