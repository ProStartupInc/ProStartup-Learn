Současné HTML elementy

<dt>

Hlavní strana > Web > Front⁠-⁠End > HTML > Elementy > <dt>

<dt> (definition term) je párový tag používaný pro vytváření termínů v definičních seznamech.

Syntaxe

<dt> je párový tag. Jde o potomka <dl>. Mezi počáteční a koncový tag se může dát např. text, tabulky, obrázky, nebo dokonce jiné seznamy.

1
2
3
<dl>
    <dt globální-atribut="hodnota">Obsah</dt>
</dl>

Element se může také nacházet v elementu <div>, který je potomkem <dl>.

1
2
3
4
5
<dl>
    <div>
        <dt globální-atribut="hodnota">Obsah</dt>
    </div>
</dl>

Atributy

<dt> podporuje jen globální atributy.

Vzhled

<dt> lze v CSS stylovat. Defaultně jde o blokový element.

1
2
3
dt {
    display: block;
}

Využití

<dt> se používá k vytváření termínů u definičních seznamů (<dl>). I když to tak nemusí vždy být, většinou se <dt> nachází před <dd>.

1
2
3
4
5
5
6
7
<dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language; značkovací jazyk používaný ve web designu</dd>
    <dt>CSS</dt>
    <dd><b>C</b>ascading <b>S</b>tyle <b>S</b>heets; jazyk používaný ke stylování HTML dokumentů</dd>
    <dt>JavaScript</dt>
    <dd>Programovací jazyk používaný pro vytváření interaktivích webových stránek</dd>
</dl>
