Současné HTML elementy

<code>

Hlavní strana > Web > Front⁠-⁠End > HTML > Elementy > <code>

<code> je párový tag používaný k reprezentaci kódu na stránce.

Syntaxe

<code> je párový tag. Mezi ten počáteční a koncový se dává "kód", jež se má uživateli zobrazit jako text.

1
<code globální-atribut="hodnota">Text</code>

Atributy

<code> podporuje jen globální atributy.

Vzhled

<code> lze v CSS stylovat. Text v něm je defaultně vykreslen ve výchozím neproporciálním (v angličtině monospace) písmu uživatelova prohlížeče.

1
2
3
4
code {
    display: inline;
    font-family: monospace;
}

Využití

<code> se využívá pro reprezentaci počítačového kódu na stránce v textové formě. Je ale třeba mít na paměti, že je stále nutné používat entity a dodržovat pravidla syntaxe HTML - pokud nechcete, aby byl text uvnitř <code> formátován podle výchozích HTML pravidel, můžete obsah v elementu ještě obalit elementem <pre>.

1
<p>Pro reprezentaci kódu v HTML dokumentu použijte <code>&lt;code&gt;</code>.</p>
