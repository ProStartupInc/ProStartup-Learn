Současné HTML elementy

<hgroup>

Hlavní strana > Web > Front⁠-⁠End > HTML > Elementy > <hgroup>

<hgroup> (heading group) je sémantický element používaný k seskupování nadpisů s jejich titulky a/nebo podnadpisy.

Syntaxe

<hgroup> je párový tag. Jeho potomky by měly být pouze <script>, nadpisy a odstavce <p>.

1
2
3
<hgroup>
    Nadpisy a/nebo odstavce a/nebo skripty.
</hgroup>

Atributy

<hgroup> podporuje pouze globální atributy.

Vzhled

<hgroup> je možné v CSS stylovat. Defaultně nemá žádný vzhled.

Využití

<hgroup> je určen na seskupování nadpisů a jejich příslušných titulků, sloganů, shrnutí, anebo podnadpisů.
