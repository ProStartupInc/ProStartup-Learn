Současné HTML elementy

<aside>

Hlavní strana > Web > Front⁠-⁠End > HTML > Elementy > <aside>

<aside> je párový tag používaný pro vytváření sekce stránky, kde se má nacházet vedlejší obsah, jež s jejím hlavním obsahem souvisí jen nepřímo. Používá se například na reklamy nebo poznámky k textu.

Syntaxe

<aside> je párový tag. Mezi tím počátečním a koncovým se nachází obsah, který se zobrazí uživateli. Tím obsahem může být tatřka cokoliv, co může být potomkem <body>.

1
<aside globální-atribut="hodnota">Obsah</article>

Atributy

<aside> nemá žádné vlastní atributy, ale podporuje ty globální.

Vzhled

<aside> lze v CSS stylovat. Defaultně nemá žádné CSS vlastnosti, které by ovlivňovaly jeho vzhled. Jde o blokový element.

1
2
3
aside {
    display: block;
}

Využití

<aside> by se měl využívat na části stránky, které jejího hlavního obsahu příliš netýkají. Mezi tato využití patří poznámky k textu, reklamy, vysvětlivky...Cokoliv, co nezapadá do hlavního obsahu stránky.
