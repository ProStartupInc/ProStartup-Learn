Současné HTML elementy

<input>

Hlavní strana > Web > Front⁠-⁠End > HTML > Elementy > <input>

<input> je element používaný pro vytváření jednotlivých prvků a polí pro uživatelský vstup ve formulářích.

Syntaxe

<input> je nepárový tag.

1
<input type="hodnota" atribut="hodnota" globální-atribut="hodnota">

Atributy

<input> má poněkud velké množství vlastních atributů. Těmi dvěmi nejdůležitějšími jsou 'type', který určuje typ vytvořeného vstupového pole, a 'name', podle kterého bude možné v databázi identifikovat, co vlastně data odeslaná uživatelem vlastně reprezentují.

Vzhled

<input> lze v CSS stylovat. S každou hodnotou ' ale přichází i jiný vzhled, proto si tu výchozí CSS vlastnosti neukážeme.

Využití

<input> se používá do formulářů jako vstupní pole, do kterých uživatel zadává data, jež budou posléze odeslána serveru a tam nějak zpracována (například uložena do databáze).

Vzhledem k tomu, že každá hodnota 'type' ovlivňuje <input> jinak, si tu moc příkladů využití neukážeme - více zjistíte na individuálních stránkách každé hodnoty (viz 'Seznam hodnot 'type').

Základní textový formulář

Základní textové formuláře lze vytvořit jen s elementy , <input> a <label>. V příkladě níže je <input> využit dvakrát, přičemž jejich atributy 'type' mají hodnoty "text" a "submit".

1
2
3
4
5
<form>
    <label for="formular">Text: </label>
    <input type="text" id="formular" name="jmeno"><br><br>
    <input type="submit">
</form>
