Současné HTML elementy

<abbr>

Hlavní strana > Web > Front⁠-⁠End > HTML > Elementy > <abbr>

<abbr> (abbreviation) je párový tag používaný k vytváření zkratek v textu.

Syntaxe

<abbr> je párový tag. Mezi tím počátečním a koncovým se nachází text, který se zobrazí uživateli.

1
2
<abbr globální-atribut="hodnota">Text, který se zobrazí uživateli</abbr>

Atributy

<abbr> nemá žádné vlastní atributy, ale podporuje ty globální.

Globální atribut 'title' má u zkratek sémantický význam - jeho hodnota je brána jako text, který je onou zkratkou reprezentován.

Vzhled

<abbr> lze v CSS stylovat. Bez CSS vypadá stejně jako jiný obyčejný text. Liší se od něj jen v jednom aspektu: když se přes něj položí kurzor, v některých prohlížečích se vedle něj objeví otazník.

1
2
3
abbr {
    cursor: help;
}

Využití

Element <abbr> se používá hlavně na zkratky v textu (např. CSS nebo OSN). Často se v nich ještě používá globální atribut 'title', jehož hodnota udává text, který je zkratkou reprezentovaný a uživateli se zobrazí, když přes ni přejede myší (př.: CSS a OSN). Zkratky s tímto atributem jsou v mnoha prohlížečích podtrženy tečkovanou čarou.

1
2
<p><abbr>HTML</abbr> je základem web designu.</p>
<p><abbr title="HyperText Markup Language">HTML</abbr> je základem web designu.</p>
1
2
<abbr> bez 'title'.
<abbr> s 'title'.
