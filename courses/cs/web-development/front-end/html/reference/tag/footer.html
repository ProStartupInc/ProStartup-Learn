Současné HTML elementy

<footer>

Hlavní strana > Web > Front⁠-⁠End > HTML > Elementy > <footer>

<footer> je element označující patičku - sekci stránky, která obsahuje například kontaktní údaje nebo informace o autorských právech.

Syntaxe

<footer> je párový tag. Pokud je jeho nejbližším rodičovským elementem nějaký nadpis nebo <hgroup>, element se vztahuje pouze na něj. Pokud ale není potomkem žádného takového tagu, patička se vztahuje na celý dokument.

1
2
3
<footer>
    Obsah.
</footer>

Atributy

<footer> podporuje pouze globální atributy.

Vzhled

<footer> lze v CSS stylovat. Níže jsou defaultní CSS vlastnosti.

1
2
3
footer {
    display: block;
}

Využití

<footer> se používá na vytváření patiček. To jsou sekce stránky, které například obsahují informace o autorských právech, autorovi, odkazy na související stránky...

Nemusí se vždy (i když tomu tak většinou je) nacházet na konci stránky nebo sekce, která specifikována nadpisy. Také může obsahovat nějaké dlouhé texty, jako třeba přílohy, rejstříky, a podobně.
