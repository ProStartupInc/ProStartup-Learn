2. Základy <iframe>

Hlavní strana > Web > Front⁠-⁠End > HTML > HTML 102 > Lekce 2

iframy (inline frame) se používají pro vkládání jiného HTML dokumentu na stránku. To se nevztahuje jenom na lokální dokumenty, ale i na ty externí - na stránku je tak možné vložit obsah z jiného webu.

Vytvoření základního iframu

iframy se vytváří s párovým tagem <iframe>. K tomu, aby byly užitečné, je potřeba ale jeden atribut - 'src', jehož hodnotou je odkaz na soubor.

Mezi počáteční a koncový tag by se měl vložit text pro uživatele, kterým se iframe z důvodu nekompatibility nenačte (např. Omlouváme se, ale váš prohlížeč nepodporuje element iframe). Tento text by ale měl být obalen nějakým elementem (např. <p>)!

Atributy <iframe>

Element <iframe> má kromě 'src' několik dalších základních atributů:

  • 'height': Určuje výšku iframu v pixelech. Hodnotou může být libovolné číslo.
  • 'width': Určuje šířku iframu v pixelech. Hodnotou může být libovolné číslo.
  • 'type': Určuje popis elementu. Užitečný pro čtečky obrazovek. Hodnotou je jakýkoliv text.
  • ''frameborder': Určuje, jestli má iframe okraje nebo ne. Má dvě možné hodnoty: "0" (žádné okraje) a "1" (okraje; výchozí hodnota). Tento atribut je již zastaralý a místo něj se používá CSS (které ale není subjektem tohoto kurzu, což je důvod, proč si ho ukazujeme).

Shrnutí

  • iframy umožňují na stránku vložit jiné lokální i externí HTML dokumenty
  • iframy se vytváří s párovým tagem <iframe>
  • K tomu, aby iframe fungoval, se musí použít atribut 'src', jehož hodnotou je cesta k HTML souboru nebo URL stránky
  • Atributy 'height' a 'width' určují výšku a šířku iframu v pixelech; jejich hodnotou může být libovolné číslo
  • Atribut 'title' definuje popis iframu po čtečky obrazovek; hodnotou je libovolný text
  • Atribut 'frameborder' určuje, zda má iframe okraje nebo ne - pokud je hodnotou "1" (výchozí), okraje budou; pokud je hodnotou "0", okraje budou chybět
