HTML

HTML (HyperText Markup Language) je značkovací jazyk používaný pro vytváření webových stránek. Dříve se v něm webové stránky i stylovaly, ale k tomuto účelu se používá CSS. HTML se v současné době používá hlavně na strukturu stránky. V kombinaci s CSS a JavaScriptem s ním lze vytvářet komplexní webové stránky.

První verze HTML (HTML 1.0) byla vytvořena v roce 1993. Jazyk ale nebyl příliš používán, protože internet nebyl ještě tak rozšířený jako dnes. Druhá verze (HTML 2.0) byla vydána roku 1995 a přinesla mnoho vylepšení a funkcí, jako třeba formuláře. Třetí verze (HTML 3.2) byla vydána v lednu 1997 a stala se tak populární, že se de facto stala standartem pro vývoj webových stránek. Mezi přidané funkce patří např. tabulky, horní a dolní indexy a applety. V prosinci stejného roku byla vydána verze HTML 4.0, která přidala podporu pro CSS. Poslední významnou verzí je HTML5, která byla vydána v roce 2014.

HTML5

Poslední velká verze HTML, která byla vydána v roce 2014 a je všeobecně příjmaným standartem pro design webových stránek. Bylo v něm přidáno mnoho sémantických elementů (např. <aside>) a v prospěch CSS odstraněno několik, které existovaly jen na stylování (např. <basefont>).
